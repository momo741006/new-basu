# 🚀 虹靈御所八字系統 - 快速參考卡

## 🎯 快速開始

### 界面選擇
- **經典版**: `http://localhost:5000/` - 適合初學者
- **Ultimate版**: `http://localhost:5000/ultimate` - 適合專業用戶

### 基本操作
1. 選擇界面版本
2. 填寫出生信息
3. 點擊生成按鈕
4. 查看分析結果

### 故障排除
1. **無法生成結果**: 檢查所有信息是否填寫完整
2. **網絡錯誤**: 確認後端服務運行正常
3. **界面異常**: 清除瀏覽器緩存重試
